# Santraspasos

App simple para generar flyers verticales de traspaso de franquicias.

## Uso

1. Abrir `index.html` en el navegador.
2. Cargar marca, inversión, ubicación y datos opcionales.
3. Subir foto de fondo y logo en PNG.
4. Ajustar color/opacidad del filtro y color de los datos.
5. Descargar el flyer en PNG o JPG.

La app no guarda datos ni imágenes. Todo se procesa en el navegador de la persona que la usa.

## Publicación recomendada

Para que el equipo la use con un link, subir estos archivos a Netlify, Vercel o GitHub Pages:

- `index.html`
- `styles.css`
- `script.js`

La opción más simple para empezar es Netlify:

1. Crear una cuenta en Netlify.
2. Entrar a "Add new site".
3. Elegir "Deploy manually".
4. Subir la carpeta completa.
5. Compartir el link generado con el equipo.

No hace falta base de datos ni backend.
